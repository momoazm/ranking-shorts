# 21st.dev Magic (standing rule)

Standing rule (Moemen): **every** website build/change must use 21st.dev Magic (`/ui`, the
`@21st-dev/magic` MCP) for premium sections (hero, bento/feature grids, marquees, pricing, CTAs),
then restyle to MOMO brand — never ship its defaults. On vanilla sites, take its structure +
interaction and reimplement. Key in `.mcp.json` / `API.env` (`MAGIC_21ST_API_KEY`). See
[web.md](../../projects/website/web.md).
