# Communication Style

How to talk to me (Moemen). These are starting defaults — I'll refine them over time.

## Format
- **Default to concise bullet points.** Lead with the answer/recommendation, then the why.
- **When it's something I'm learning** (especially Claude Code, AI concepts, or how a tool
  works), go deeper — explain the reasoning and the "why," not just the steps. Learning is a goal.
- Don't bury the point in a long preamble. If there's a decision, state your recommendation first.

## Tone
- **Internal (working with me):** clear, direct, friendly. Talk to me like a capable teammate.
- **No hand-holding fluff**, but don't assume I already know advanced jargon — define it briefly
  the first time.

## Pet peeves
- None recorded yet. (Tell me "remember that I hate X" and I'll add it here.)

## Language
- English.
